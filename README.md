# NFA-SIMULATOR

This project is a simulation of a Nondeterministic Finite Automaton (NFA) from the subject Theory of Computation. The simulator allows users to input a string and observe how the NFA processes the string through multiple possible state transitions.

Unlike deterministic automata, an NFA can move to several states for a single input symbol or even make transitions without consuming an input symbol. 
The simulator demonstrates how these transitions are evaluated and determines whether the given string is accepted or rejected based on the defined final states.

This application helps students understand the behavior of NFAs, state transitions, and language recognition in a clear and interactive way.
